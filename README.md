# Student Grade Tracker

A Python console application for managing student records and grades.

## Features

* Add students to the class roster
* View all students
* Search for a student by name
* Update student grades
* Remove students from the roster
* Calculate class average grade
* Sort students alphabetically
* Sort students by grade
* Display the highest grade in the class
* Display the lowest grade in the class
* Save student data to a text file
* Load student data from a text file

## Concepts Practiced

* Object-Oriented Programming (Classes and Objects)
* Class Methods
* Lists of Objects
* File I/O
* Data Persistence
* Searching and Sorting
* Loops and Conditional Logic
* Functions and Program Organization
* Git and GitHub
